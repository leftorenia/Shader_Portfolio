# No.01 透明なシェーダを作る

https://nn-hokuson.hatenablog.com/entry/2016/10/05/201022

## 元Shaderの概要

Surface Shader を用いて半透明のオブジェクトを描画するシンプルなシェーダ。ユーザーはアルベド色とアルファ値を設定するだけで簡易的な半透明表現を実現できる。

## 設計方針

ライティングや影などの機能は実装しない
色と透明度のみを出力するUnlitシェーダとして実装

## その他

Surface Shaderでは、環境光（Ambient）や各種ライト（Directional Light、Point Lightなど）の影響を考慮したライティング計算がUnityによって自動生成される。一方、本シェーダではライティング処理を意図的に省略し、最終色をフラグメントシェーダから直接出力する構成とした。
