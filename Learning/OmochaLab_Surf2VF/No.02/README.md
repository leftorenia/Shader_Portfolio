# No.02 氷のような半透明シェーダを作る

https://nn-hokuson.hatenablog.com/entry/2016/10/07/221724

## 元Shaderの概要

Surface Shader を用いて、視線ベクトル(viewDir)とワールド法線の内積からアルファ値を計算するシェーダ。視線と正対する面ほど透明に、縁に近づくほど不透明になり、氷のような半透明表現を実現できる。

## 設計方針

ライティングや影などの機能は実装しない
No.01と同様、色と透明度のみを出力するUnlitシェーダとして実装
アルファ計算に必要なワールド法線と視線ベクトルは頂点シェーダで計算して渡し、フラグメントシェーダで正規化し直して使う

## メモ

法線のワールド変換は UnityObjectToWorldNormal() を使う。unity_ObjectToWorld を直接掛けると非一様スケール時に法線が歪む。

視線ベクトルは UnityWorldSpaceViewDir(ワールド座標) = カメラ位置 − ワールド座標。面からカメラへ向かう向きになる。

頂点シェーダで正規化しても、ラスタライズの補間を経ると単位長からずれるため、フラグメント側で normalize し直す必要がある。

alpha*1.5 は1を超えることがある。通常(LDR)のレンダーターゲットでは出力時に0〜1へクランプされるので元記事と同じ見た目になる。
