# Shader_Portfolio

本リポジトリは、ポートフォリオとして公開しているUnity Shader集です。

シェーダー表現の検証や技術の習得を目的として制作したものを掲載しています。


## 開発環境

- Unity
- ShaderLab / HLSL

## 参考資料

OmochaLab_Surf2VFフォルダ内のShaderは、以下の技術記事を参考に実装しています。

- おもちゃラボ「Unityシェーダ入門」シリーズ
  - https://nn-hokuson.hatenablog.com/entry/2018/02/15/140037

参考資料のアルゴリズムや実装方針を学習目的で参照し、Surface Shaderで実装されている内容をVertex/Fragment Shaderへ書き換えた上で、機能追加や実装方法の変更を行っています。

各ShaderのREADMEには、参考にした記事や独自に実装・改良した箇所を個別に記載しています。
